JDK(Java Devlopement Kit):

JDK is a software Devlopement Environment used for making applets and Java Applications.Java Devlopers can use it on Windows, Macos And Linux. JDK help them to code and run java programs. It is also possible to install more than one JDK version on the same Computer

Block Diagram of JDK:

The JDK contains a private java virtual machine(JVM) and a few other resources such as an interpreter/ loder(JAVA), a compiler(javaC), and archiver(jar), a Documentation generator(javadoc), etc. To complete the Devlopement of a java application

KeyPoints of JDK:

    Here are some important JDK keypoints

  • JDK contains tools required to write java programs and JRE to execute them
  • It includes as a compiler, java application launcher, Appletviewer, etc.
  • It includes a compiler thats converts codes written in java into byte code for virtual machine

JRE(Java Runtime Environment)

JRE is an acronym for java Runtime Environment.It is also Written as JAVA RTE. The JRE is a set of software tools which are used for Devloping java Applications. It contains the class libraries. loader class and JVM. In simple terms, if you want to run a java programs you need JRE. if you are note a programmer you don't need to install JDK, but just JRE to run Java programs
